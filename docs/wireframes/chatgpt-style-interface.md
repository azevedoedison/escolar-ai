## 9. RESPONSIVIDADE

### Mobile (< 768px)
- Sidebar oculta (hamburger menu)
- Chat em tela cheia
- Categorias em scroll horizontal
- Teclado otimizado para perguntas

### Tablet (768px - 1024px)
- Sidebar colapsável
- Chat + sidebar lado a lado
- Categorias em grid 3x2

### Desktop (> 1024px)
- Layout completo (sidebar + chat)
- Sidebar fixa
- Chat centralizado

---

**Design Principles:**
1. **Familiar:** Inspirado no ChatGPT (usuário já sabe usar)
2. **Seguro:** Visual claro de quando conteúdo é bloqueado
3. **Amigável:** Emojis, cores suaves, linguagem simples
4. **Educacional:** Categorias por matéria, sugestões de perguntas
5. **Transparente:** Pais têm visibilidade total
